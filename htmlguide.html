1 User guide

In this section we describe the application we developed from a user perspective. Section 1.1 discusses how the application can be set up and started. Section 1.2 discusses how to change some global options. Section 1.3 explains how the virtual camera can be controlled by the user. Section 1.4 describes how objects can be added to the environment. Section 1.5 describes the interaction with objects by the user. Finally, Section 1.6 discusses saving to and loading from files.

In this guide, we will occasionally refer to menus and menu items within them. These menus (File, Edit, etc.) can be found in the menu bar at the top of the application. Menus and menu items are referred to in bold text. A menu item will be referred to by first stating the name of the menu, followed by a right arrow, and then the name of the menu item. For instance, File → Load... refers to the Load... item in the File menu. Finally, many options can also be activated through a keyboard shortcut. These will be between brackets and emphasized.

1.1 Preliminaries

Building the project from source can be done using maven. We recommend using an IDE, such as NetBeans, for this. This will retrieve some required libraries (specifically OpenGL support through JOGL) and build the entire application. Upon starting the application, an empty environment will be shown. An example of this can be found in Figure 1.

The environment shown here consists of a floor and four walls surrounding it, but it does not contain any vehicles or other object. Adding these is discussed in Section 1.4.
